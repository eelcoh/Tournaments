**viewInstallBanner function:**

```elm
viewInstallBanner : Model Msg -> Element.Element Msg
viewInstallBanner model =
    case model.installBanner of
        BannerHidden ->
            Element.none

        BannerShowingIOS ->
            bannerRow
                [ Element.el
                    [ Font.color Color.white
                    , UI.Font.mono
                    , Font.size (UI.Font.scaled 0)
                    ]
                    (Element.text "[ add to home screen ] -- tap \u{2197} then \"Add to Home Screen\"")
                , dismissButton
                ]

        BannerShowingAndroid ->
            bannerRow
                [ Element.el
                    [ Element.Events.onClick TriggerAndroidInstall
                    , Element.pointer
                    , Font.color Color.orange
                    , UI.Font.mono
                    , Font.size (UI.Font.scaled 0)
                    ]
                    (Element.text "[ Installeer App ]")
                , dismissButton
                ]
```

Where `bannerRow` wraps children in the terminal-styled row:

```elm
bannerRow : List (Element.Element Msg) -> Element.Element Msg
bannerRow children =
    Element.row
        [ Element.width Element.fill
        , Element.paddingXY 12 6
        , Element.spacing 8
        , Background.color Color.black
        , Border.widthEach { top = 1, bottom = 0, left = 0, right = 0 }
        , Border.color Color.terminalBorder
        ]
        children
```

And `dismissButton` is:

```elm
dismissButton : Element.Element Msg
dismissButton =
    Element.el
        [ Element.alignRight
        , Element.Events.onClick DismissInstallBanner
        , Element.pointer
        , Font.color Color.grey
        , UI.Font.mono
        , Font.size (UI.Font.scaled 0)
        ]
        (Element.text "[x]")
```

Note: `\u{2197}` is the ↗ arrow character in Elm string syntax (Unicode escape). Use this to avoid any encoding issues.

**Update the `body` layout in `view`:**

The current `body` is:
```elm
body =
    Element.layout
        (UI.Style.body
            [ Element.inFront
                (Element.el
                    [ Element.alignBottom, Element.width Element.fill ]

                )
            ]
        )

```

Update to stack the install banner above the status bar. Both use `alignBottom` and `inFront`. Wrap both in a single `inFront` column so banner renders above status bar:

```elm
body =
    Element.layout
        (UI.Style.body
            [ Element.inFront
                (Element.column
                    [ Element.alignBottom, Element.width Element.fill ]
                    [ viewInstallBanner model
                    , viewStatusBar model
                    ]
                )
            ]
        )
        page
```

When banner is `BannerHidden`, `viewInstallBanner` returns `Element.none` so only the status bar renders — no layout shift.

**Elm style notes:**
- Color.black is `rgb255 13 13 13` — dark background consistent with terminal style.
- `UI.Font.mono` sets Sometype Mono font — consistent with status bar and rest of app.
- `UI.Font.scaled 0` is 16px — same as status bar text.
- The `↗` arrow is Unicode U+2197. In Elm strings use `"\u{2197}"` syntax.
