  <name>Task 1: Restyle commentBox and blogBox with resultCard treatment</name>
  <files>src/Activities.elm</files>
  <action>
Restyle `commentBox` and `blogBox` in `src/Activities.elm`:

**commentBox** — replace the current `column [ paddingXY 0 8, width fill, Border.widthEach { bottom = 1 ... }, ... ]` wrapper with `column (UI.Style.resultCard [ Screen.className "commentBox" ])`. Inside the card, add `paddingXY 12 8` to the header row (timestamp + author) and body paragraph. The timestamp `[HH:MM]` gets `Font.size 12` (it likely already has `Font.color Color.grey` — keep it). Author label keeps `Font.color Color.orange`.

**blogBox** — same `resultCard` base, but add a left border accent using `Border.widthEach { left = 3, right = 1, top = 1, bottom = 1 }` and `Border.color Color.activeNav` (replacing the `Border.color Color.terminalBorder` from resultCard — use `Element.el` wrapper or override in the attrs list passed to resultCard). The simplest approach: pass the accent override directly in the attrs list to `resultCard`:

```elm
blogBox author title blog tz dt =
    column
        (UI.Style.resultCard
            [ Screen.className "blogBox"
            , Border.widthEach { left = 3, right = 1, top = 1, bottom = 1 }
            , Border.color Color.activeNav
            ]
        )
        [ row [ paddingXY 12 8, Element.spacing 8 ]
            [ Element.el [ Font.color Color.grey, Font.size 12, UI.Font.mono ] (Element.text (UI.Text.timeText tz dt))
            , Element.el [ Font.color Color.orange, UI.Font.mono ] (Element.text ("## " ++ title))
            ]
        , Element.el [ paddingXY 12 8, width fill ] (blogView blog)
        , Element.el [ alignRight, paddingXY 12 8, Font.color Color.grey, Font.size 12, UI.Font.mono ]
            (Element.text ("-- " ++ author ++ ", " ++ UI.Text.dateText tz dt))
        ]
```

Similarly for commentBox:
```elm
commentBox author comment tz dt =
    column
        (UI.Style.resultCard [ Screen.className "commentBox" ])
        [ row [ paddingXY 12 8, Element.spacing 8 ]
            [ Element.el [ Font.color Color.grey, Font.size 12, UI.Font.mono ] (Element.text (UI.Text.timeText tz dt))
            , Element.el [ Font.color Color.orange, UI.Font.mono ] (Element.text (author ++ ":"))
            ]
        , Element.el [ paddingXY 12 8, width fill ] (commentView comment)
        ]
```

Note: `resultCard` sets `paddingXY 0 0` so inner rows handle their own padding — consistent with the match row pattern from Phase 22.

Note on blog left accent: elm-ui applies `Border.widthEach` as a style attribute. When you pass it in the attrs list to `resultCard`, it will override the resultCard's `Border.width 1`. This is the correct approach — no wrapper needed.
