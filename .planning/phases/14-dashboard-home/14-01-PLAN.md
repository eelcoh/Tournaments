- Each row is a clickable `Element.el [onClick (NavigateTo idx), pointer, height (px 44)]` wrapping an `Element.row`

- Completion functions:

- Header: `UI.Text.displayHeader "overzicht"` (renders `--- OVERZICHT ---`)
- Wrap content in `UI.Page.page "dashboard" [...]` for consistent padding
