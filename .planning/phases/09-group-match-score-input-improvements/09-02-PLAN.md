**B. Rewrite `viewInput` to add flag images:**

The current signature `viewInput _ matchID homeTeam awayTeam mScore` is fine. Restructure the body to add 24px flag images:

```elm
viewInput : a -> MatchID -> Team -> Team -> Maybe Score -> Element.Element Msg
viewInput _ matchID homeTeam awayTeam mScore =
    let
        -- flag image helper (24px)
        flagImg team =
            Element.image
                [ Element.height (Element.px 24)
                , Element.width (Element.px 24)
                , Element.centerY
                ]
                { src = T.flagUrl (Just team)
                , description = T.display team
                }

        makeAction act val =
            case String.toInt val of
                Just v -> act v
                Nothing -> NoOp

        inputField v act =
            Input.text
                (UI.Style.scoreInput
                    [ width (px 60)
                    , Border.rounded 0
                    , Element.htmlAttribute (Html.Attributes.attribute "inputmode" "numeric")
                    ]
                )
                { onChange = makeAction act
                , text = v
                , label = Input.labelHidden ".."
                , placeholder = Just (Input.placeholder [] (Element.text v))
                }

        wrap fld =
            Element.el (UI.Style.wrapper [ centerX, centerY ]) fld

        extractScore extractor =
            mScore
                |> Maybe.andThen extractor
                |> Maybe.map String.fromInt
                |> Maybe.withDefault ""

        homeInput = inputField (extractScore S.homeScore) (UpdateHome matchID) |> wrap
        awayInput = inputField (extractScore S.awayScore) (UpdateAway matchID) |> wrap

        homeBadge =
            Element.el [ Font.color Color.white, UI.Font.mono, Font.size (UI.Font.scaled 1), centerY ]
                (Element.text (T.display homeTeam))

        awayBadge =
            Element.el [ Font.color Color.white, UI.Font.mono, Font.size (UI.Font.scaled 1), centerY ]
                (Element.text (T.display awayTeam))
    in
    Element.row (UI.Style.activeMatch [ centerX, paddingXY 4 16, spacing 8 ])
        [ flagImg homeTeam
        , homeBadge
        , homeInput
        , Element.el [ Font.color Color.grey, UI.Font.mono, centerY ] (Element.text "-")
        , awayInput
        , awayBadge
        , flagImg awayTeam
        ]
```

Note: remove the `>` prefix el from the row (the flags now provide visual anchoring). The `>` prompt is no longer needed as a prefix since flags make the active match visually clear.
