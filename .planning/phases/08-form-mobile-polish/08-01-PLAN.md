**Part A — View.elm: add `viewFormNavBar` and wire into inFront column**

1. Add `viewFormNavBar : Model Msg -> Element.Element Msg` function that produces a 48px fixed bottom bar shown only when `model.app == Form`. When app is not Form, returns `Element.none`.

   Bar layout (full width, dark background, terminal border top):
   ```
   [ < vorige ]  [ stap 2/6 · 12 open ]  [ volgende > ]
   ```

   Style: Same as `viewStatusBar` — `Background.color Color.black`, `Border.widthEach { top = 1, bottom = 0, left = 0, right = 0 }`, `Border.color Color.terminalBorder`, `Element.paddingXY 12 0`, `Element.height (Element.px 48)`.

2. Nav button semantics:
   - `prev = Basics.max (model.idx - 1) 0`
   - `next = Basics.min (model.idx + 1) (List.length model.cards - 1)`
   - At first card (`model.idx == 0`): vorige button uses `UI.Style.Irrelevant` semantics (greyed) and fires `NoOp`
   - At last card (`model.idx == List.length model.cards - 1`): volgende button uses `UI.Style.Irrelevant` and fires `NoOp`
   - Otherwise both use `UI.Style.Focus` semantics (existing pill style) and fire `NavigateTo prev` / `NavigateTo next`

   Button format — monospace text, 44px touch target, using existing `UI.Button.pill` or inline with:
   ```elm
   Element.el
       [ Element.Events.onClick msg
       , Element.pointer
       , Element.height (Element.px 48)
       , Element.centerY
       , Font.color (if disabled then Color.grey else Color.orange)
       , UI.Font.mono
       ]
       (Element.text label)
   ```

3. Center info: compute `cardCenterInfo model` helper that returns `String`:
   - Uses `model.idx`, `model.cards`, `model.bet`
   - Step portion: `"stap " ++ String.fromInt (model.idx + 1) ++ "/" ++ String.fromInt (List.length model.cards)`
   - Incomplete count: detect current card type and compute:

     **GroupMatchesCard**: count = incomplete match count from `bet.answers.matches`:
     ```elm
     let
         openCount =
             List.filter (\(_, gm) -> not (Form.GroupMatches.isMatchComplete gm)) model.bet.answers.matches
                 |> List.length
     ```
     Wait — `Form.GroupMatches` does not expose `isMatchComplete`. Use the lower-level:
     ```elm
     import Bets.Types.Answer.GroupMatch as GroupMatch
     openCount = List.length model.bet.answers.matches
                 - List.length (List.filter (GroupMatch.isComplete << Tuple.second) model.bet.answers.matches)
     ```
     Label: `"stap N/M · " ++ String.fromInt openCount ++ " wedstrijden open"` (if openCount > 0), else `"stap N/M [x]"`

     **BracketCard**: extract BracketState from model.cards using pattern match. Count incomplete SelectionRounds:
     ```elm
     let
         bracketCard = List.head (List.filter (\c -> case c of BracketCard _ -> True; _ -> False) model.cards)
     in
     case bracketCard of
         Just (BracketCard { bracketState }) ->
             case bracketState of
                 BracketWizard { selections } ->
                     let
                         rounds = [ (ChampionRound, 1), (FinalistRound, 2), (SemiRound, 4),
                                    (QuarterRound, 8), (LastSixteenRound, 16), (LastThirtyTwoRound, 32) ]
                         openRounds = List.filter (\(r, cap) ->
                             List.length (roundTeams r selections) < cap) rounds |> List.length
                     in
                     if openRounds == 0 then "stap N/M [x]"
                     else "stap N/M · " ++ String.fromInt openRounds ++ " ronden open"
     ```
     Import `Form.Bracket.Types as BracketTypes` and use `BracketTypes.roundTeams`, `BracketTypes.BracketWizard`.

     **TopscorerCard**: `if Form.Topscorer.isComplete model.bet then "stap N/M [x]" else "stap N/M · 1 open"`

     **ParticipantCard**: `if Form.Participant.isComplete model.bet then "stap N/M [x]" else "stap N/M · gegevens open"`

     **IntroCard / SubmitCard**: just `"stap N/M"` (no count, these are informational)
