The new function:
```elm
dataRow : ButtonSemantics -> msg -> List (Element.Element msg) -> Element.Element msg
dataRow semantics msg children =
    Element.row
        (Style.button semantics
            [ Element.padding 8
            , Element.width Element.fill
            , onClick msg
            , Element.pointer
            ]
        )
        children
```

Use `UI.Style.Potential` as the recommended semantics for neutral data rows that navigate on click (orange hover = clear interactive affordance). This is a convention documented in the plan; callers choose their own semantics.
