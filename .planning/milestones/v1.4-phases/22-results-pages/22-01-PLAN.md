**`displayMatches`**: Change from `Element.wrappedRow` to a grouped column. Group the `List MatchResult` by `.group` using `List.Extra.groupBy` (already available via elm-community/list-extra). For each group, render a section header followed by a card containing the match rows:

```elm
displayMatches : Access -> List MatchResult -> Element.Element Msg
displayMatches access matches =
    let
        groups =
            List.Extra.groupBy (\a b -> a.group == b.group) matches

        viewSection grpMatches =
            case grpMatches of
                [] ->
                    Element.none

                first :: _ ->
                    let
                        hdr =
                            UI.Text.displayHeader ("Groep " ++ Bets.Types.Group.toString first.group)

                        rows =
                            List.map (displayMatch access) grpMatches
                    in
                    Element.column
                        (UI.Style.resultCard [ Element.spacing 0 ])
                        (hdr :: rows)
    in
    Element.column
        [ Element.spacing 16, Element.width Element.fill ]
        (List.map viewSection groups)
```

**`displayMatch`**: Replace the 160×100px tile with a full-width horizontal row using `viewTeamSmallHorizontal`:
