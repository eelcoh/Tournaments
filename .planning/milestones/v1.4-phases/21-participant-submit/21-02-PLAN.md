   ```elm
   filledMatches = bet.answers.matches |> List.filter GroupMatch.isComplete |> List.length
   gmValue = String.fromInt filledMatches ++ "/48"
   gmColor = if Form.GroupMatches.isComplete bet then Color.green else Color.red
   ```

2. **Knock-out schema** — label `"knock-out schema"`:
   ```elm
   bracketComplete = Form.Bracket.isCompleteQualifiers bet
   bracketValue = if bracketComplete then "volledig" else "onvolledig"
   bracketColor = if bracketComplete then Color.green else Color.red
   ```

3. **Topscorer** — label `"topscorer"`:
   ```elm
   topscorerComplete = Form.Topscorer.isComplete bet
   topscorerValue = case bet.answers.topscorer of
       Answer (Just name, _) _ -> name
       _ -> "—"
   topscorerColor = if topscorerComplete then Color.green else Color.red
   ```
   Note: `bet.answers.topscorer` is of type `AnswerTopscorer = Answer Topscorer Points`; `Topscorer = (Maybe String, Maybe Team)`. Pattern match using `Answer topscorer _` then `case topscorer of (Just name, _) -> name; _ -> "—"`.

4. **Naam** — label `"naam"`:
   ```elm
   naamValue = StringField.toString bet.participant.name |> Maybe.withDefault "—"
   naamColor = case bet.participant.name of Changed _ -> Color.green; _ -> Color.red
   ```
   Use `StringField.toString` if it exists, otherwise pattern match: `case bet.participant.name of Initial s -> if s == "" then "—" else s; Changed s -> s; Error s -> s`.

5. **E-mail** — label `"e-mail"`:
   Same pattern as Naam but for `bet.participant.email`, color green only if `Changed _`, not `Error _`.
