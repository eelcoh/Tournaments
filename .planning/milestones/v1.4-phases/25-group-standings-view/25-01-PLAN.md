  <action>
Create a new Elm module `Results.GroupStandings` that:

1. **Defines a local TeamStanding record:**
```elm
type alias TeamStanding =
    { team : Bets.Types.Team
    , played : Int
    , won : Int
    , drawn : Int
    , lost : Int
    , goalsFor : Int
    , goalsAgainst : Int
    , points : Int
    }
```

2. **Computes standings from MatchResults:**

Write `computeStandings : List MatchResult -> List ( Group, List TeamStanding )` that:
- Groups MatchResult list by `.group` using `List.Extra.groupWhile (\a b -> a.group == b.group)`
- For each group, collects all teams from home/away fields, then for each match with a set score (`Just (Just h, Just a)`), accumulates wins/draws/losses/goals into a `Dict String TeamStanding` keyed by `team.teamID`
- Points: win = 3, draw = 1, loss = 0
- Sorts teams within each group by points descending (use `List.sortBy (\s -> -s.points)`)
- Returns list of `(Group, List TeamStanding)` pairs

3. **Position color helper:**
```elm
positionColor : Int -> Element.Attribute msg
positionColor pos =
    case pos of
        1 -> Font.color UI.Color.green
        2 -> Font.color UI.Color.green
        3 -> Font.color UI.Color.orange
        _ -> Font.color UI.Color.white
```
(pos is 1-indexed rank within the group)

4. **viewStandingRow : Int -> TeamStanding -> Element.Element Msg**

A row displaying: position dot, team code, played, won, drawn, lost, goals for, goals against, points.
- Uses `Element.paddingXY 12 6`, `Element.width Element.fill`
- Bottom border: `Border.widthEach { bottom = 1, top = 0, left = 0, right = 0 }`, `Border.color UI.Color.terminalBorder`
- Apply `positionColor pos` to the entire row via `Font.color` on the row element
- Column layout hint: use `Element.row [ Element.width Element.fill, Element.spaceEvenly ]` with fixed-width `Element.el [ Element.width (Element.px N) ]` for alignment
- Suggested columns (left to right): pos (px 20), team code (px 60, `T.display team`), P played (px 24), W (px 24), D (px 24), L (px 24), GF (px 24), GA (px 24), Pts (px 28, bold)

5. **viewGroupStandings : ( Group, List TeamStanding ) -> Element.Element Msg**

```elm
viewGroupStandings ( grp, standings ) =
    let
        hdr = UI.Text.displayHeader ("Groep " ++ Bets.Types.Group.toString grp)
        rows = List.indexedMap (\i s -> viewStandingRow (i + 1) s) standings
    in
    Element.column
        (UI.Style.resultCard [ Element.spacing 0 ])
        (hdr :: rows)
```

6. **view : Model Msg -> Element.Element Msg**

```elm
view model =
    let
        hdr = UI.Text.displayHeader "Groepsstand"
        mainView =
            case model.matchResults of
                Success results ->
                    let groups = computeStandings results.results
                    in Element.column [ Element.spacing 16, Element.width Element.fill ]
                        (List.map viewGroupStandings groups)
                NotAsked -> Element.text "nog niet opgevraagd"
                Loading  -> Element.text "aan het ophalen..."
                Failure _ -> UI.Text.error "oei, oei, oei"
    in
    UI.Page.container model.screen "groepsstand"
        [ Element.column
            [ Element.paddingXY 0 8
            , Element.width Element.fill
            , Border.widthEach { bottom = 1, top = 0, left = 0, right = 0 }
            , Border.color UI.Color.terminalBorder
            ]
            [ hdr ]
        , mainView
        ]
```

**Note on score extraction:** MatchResult.score is `Maybe Bets.Types.Score` where `Score = Maybe (Maybe Int, Maybe Int)`. A set score is `Just (Just homeGoals, Just awayGoals)`. When score is Nothing or has Nothing inner values, treat match as not yet played (skip it in standings computation but still include the team in the initial team set).

**Initial team set:** Build the team set by scanning all MatchResult records in the group for homeTeam and awayTeam — each team appears in 3 matches so will be found. Initialize each TeamStanding with 0 for all fields. Then fold over matches that have a complete score to accumulate stats.

Use `Dict String TeamStanding` (keyed by `team.teamID`) for accumulation, then `Dict.values` to get the list.
