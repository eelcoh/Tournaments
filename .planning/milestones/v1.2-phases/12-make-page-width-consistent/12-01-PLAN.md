<objective>
Make the outer page column in View.elm constrained to 600px so that the top nav bar, content area, and version footer share the same width boundary as inner content views.

Purpose: Currently the top-level page column has no max-width cap, causing the nav bar and footer to stretch full viewport width on desktop while inner content is already capped. This creates visual misalignment — wider nav over narrower content.

Output: Two targeted edits — `UI.Screen.maxWidth` returns 600 (fixed constant), and `View.elm` page column gains `Element.width (Element.fill |> Element.maximum 600)`. All downstream callers (UI.Page.container, Form/View.elm) automatically inherit the correct cap.
</objective>
