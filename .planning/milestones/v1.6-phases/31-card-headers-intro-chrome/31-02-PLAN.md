Add a `viewRoundBadge` function to `Form/Bracket/View.elm` that renders the prototype `.round-badge` component:

```elm
viewRoundBadge : String -> String -> String -> Element Msg
viewRoundBadge title subtitle counter =
    Element.column
        [ Element.width Element.fill
        , Border.width 1
        , Border.color Color.activeNav
        , Background.color (Element.rgba255 0xF0 0xA0 0x30 0x0D)
        , Element.paddingXY 16 10
        , spacing 3
        ]
        [ Element.el
            [ Font.color Color.activeNav
            , UI.Font.mono
            , Font.size 11
            , Font.letterSpacing 2.2
            ]
            (Element.text (String.toUpper title))
        , Element.el
            [ Font.color Color.grey
            , UI.Font.mono
            , Font.size 10
            ]
            (Element.text subtitle)
        , Element.el
            [ Font.color Color.grey
            , UI.Font.mono
            , Font.size 10
            ]
            (Element.text counter)
        ]
```

Notes:
- `Element.rgba255 0xF0 0xA0 0x30 0x0D` — 0x0D = 13/255 ≈ 0.05, matching prototype `rgba(240,160,48,0.05)`
- `Font.letterSpacing 2.2` — prototype says 0.2em at 11px = 2.2px
- `String.toUpper title` — matches `.sec-head` pattern of uppercase display
- Three children: title (11px active), description subtitle (10px dim), counter (10px dim)
- `Element.rgba255` requires no new import — already available via `Element` module

Then update the `header` binding in `viewRoundSection` to conditionally use `viewRoundBadge` for the active round and `UI.Text.displayHeader` for inactive rounds:

```elm
header =
    if isActive then
        viewRoundBadge (roundTitle round) (roundDescription round) counterText
    else
        Element.column [ spacing 4 ]
            [ Element.row [ spacing 8 ]
                [ UI.Text.displayHeader (roundTitle round)
                , Element.el [ Font.color Color.grey, UI.Font.mono, Font.size 10 ] (Element.text counterText)
                ]
            ]
```

For inactive rounds, drop the `description` element from the header — it is only shown for the active round (in the badge). This keeps inactive sections compact.

Also remove the now-redundant `description` let binding from `viewRoundSection` since the description text is now passed directly to `viewRoundBadge`.
