In `viewProgressRail`, the `viewSegment` function currently takes `(i, _)` — the `_` ignores the card. Change the signature to `viewSegment i card` and use `cardLabel card` (already defined at line 193) for the label text.

Replace the current `viewProgressRail` implementation with:

```elm
viewProgressRail : Model Msg -> Int -> Element.Element Msg
viewProgressRail model currentIdx =
    let
        viewSegment i card =
            let
                barColor =
                    if i == currentIdx then
                        Color.activeNav

                    else if i < currentIdx then
                        Color.green

                    else
                        Color.grey

                labelColor =
                    if i == currentIdx then
                        Color.activeNav

                    else if i < currentIdx then
                        Color.green

                    else
                        Color.grey

                alphaAttrs =
                    if i > currentIdx then
                        [ Element.alpha 0.35 ]

                    else
                        []

                barAttrs =
                    [ Element.width Element.fill
                    , Element.height (Element.px 2)
                    , Background.color barColor
                    , Element.Events.onClick (NavigateTo i)
                    , Element.pointer
                    ]
                        ++ alphaAttrs

                labelAttrs =
                    [ Element.width Element.fill
                    , Font.size 8
                    , Font.letterSpacing 0.12
                    , Font.center
                    , Font.color labelColor
                    , UI.Font.mono
                    , Element.Events.onClick (NavigateTo i)
                    , Element.pointer
                    , Element.paddingEach { bottom = 3, top = 0, left = 0, right = 0 }
                    ]
                        ++ alphaAttrs
            in
            Element.column
                [ Element.width (Element.fillPortion 1) ]
                [ Element.el labelAttrs (Element.text (cardLabel card))
                , Element.el barAttrs Element.none
                ]
    in
    Element.row
        [ Element.width Element.fill
        , Element.spacing 2
        , Element.paddingXY 0 4
        ]
        (List.indexedMap viewSegment model.cards)
```

Note: `cardLabel` is already defined in the same module at line 193. The bar height changes from 3px to 2px to match the prototype `.p-bar` (height: 2px).
